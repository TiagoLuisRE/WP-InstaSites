== Installation ==

1. Copy the bookt-api folder to /wp-content/plugins
2. Go to Dashboard and click on "InstaSite" menu button which now appears in the dashboard navigation
3. Fill in your Bookt/Instamanager API key and save.
4. Go to Instasite > Data Sync and click on "Create Default Pages"

== Frequently Asked Questions ==

= Where do I get a Bookt/Instamanager API Key? =

You must be an active Bookt/Instamanager client.  Existing clients may contact support@bookt.com for assistance.  Prospective clients should contact sales@bookt.com to learn more about our products and services.

= Where can I get Bookt-approved themes? =

Bookt's premium themes provide the best integration experience with the Bookt API InstaSites plugin.  These help simplify the setup process and make it easy to control layouts and styles.

If you are an InstaSites client and we are hosting our site for you, then you will already have access to our premium themes.  If you are hosting this in your own environment please contact support@bookt.com for information about using our premium themes.
